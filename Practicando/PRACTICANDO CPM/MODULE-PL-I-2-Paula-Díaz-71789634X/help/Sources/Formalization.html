Formalization

You will see another form to fill:



  • The name of the user
  • The surname of the user
  • The email of the user
  • Any comment that the user wants to make

The mandatory information to be filled: name, surname and email

The email must be in the form: something@something.something


You can click:

  • Formalize button: It will be enabled when all the mandatory information is filled, and it will go to the final window to just end the process (when it's closed the app is initialized)

  • Back button: It will go to the warning warning before formalizing, and all the information of the formalization will be lost

When formalizing the reservation will be saved, and the discount used eliminated
