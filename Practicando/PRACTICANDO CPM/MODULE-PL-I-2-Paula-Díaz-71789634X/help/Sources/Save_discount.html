Saving the discount

After the game ends, the following window will be shown:



  • It will tell the you what discount have you won, or if you didn't won a discount
  • It would tell you if you could get a better discount or not
  • If you gained a discount, it would show you:
    • A text field to enter your id
    • It would tell you if you could get a better discount or not
    • If you gained a discount, it would show you:
      • A text field to enter the id to save
      • A checkbox to ensure that the user is 18 or older

You will be able to

  • Save when an id is introduced and the checkbox is marked.
    • If the id has already a discount assigned, the new discount will be lost
    • Else, the discount will be saved
  • Play again when the discount gained is not the maximum one. The game is initialized again

When the user wants to leave without saving the discount, a window will be shown to confirm that the user doesn't want to save the discount
